**GanD — ToDoList**

Простий та швидкий ToDoList, створений на React + TailwindCSS без збірників та залежностей. Усі компоненти працюють прямо в браузері через CDN.

**Функціонал**

**1. Списки задач**

Є стандартний список Inbox.

Користувач може створювати власні списки.

Кількість активних задач відображається біля кожного списку.

Для кожної задачі зберігається: назва, нотатки, дата, пріоритет, теги та підзадачі.

**2. Смарт-списки**

Система автоматично формує три розумні категорії:

My Day — задачі на сьогодні.

Planned — задачі з датою виконання.

Important — задачі з високим пріоритетом.

У кожному смарт-списку видно реальну кількість відповідних задач.

**3. Додавання та редагування задач**

Швидке створення задачі через поле вводу.

Редагування назви, нотаток, тегів.

Пріоритети (none / low / medium / high).

Дата виконання, час нагадування, повторення задач.

Підзадачі та перемикання їх статусу.

**4. Фільтри та пошук**

Пошук по назві, нотатках або тегах.

Фільтрація задач автоматично залежно від поточного списку.

**5. Темна / світла тема**

Перемикається вручну.

Стан зберігається у localStorage.

Tailwind працює у режимі darkMode: "class".

**6. Збереження у LocalStorage**

Усі задачі, списки, тема та налаштування зберігаються локально та не зникають при оновленні сторінки.
